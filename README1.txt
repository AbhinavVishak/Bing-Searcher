This Java application takes a search query as input, and prints the title and URL of the first web result.

Required files: startpoint.java, dep.jar

How to compile:
1.cd to the directory containing the above files in terminal/cmd.
2.run the following command <javac -cp dep.jar startpoint.java>

You should now have the Entry.class file in the same folder.

How to run:
1.cd to the directory containing the above files in terminal/cmd.
2. run the command <java -cp ".;dep.jar" Entry> in Windows, <java -cp ".:dep.jar" Entry> in Linux.
